Ducky Derby: Heather Joy 💫🌻🌙 Challenge
A high-speed, 50-level 3D rubber duck racing game built with Three.js. Inspired by Mario Kart Wii, this "Hydro-Slide" racer pits you against aggressive AI ducks in procedurally generated river environments.
✨ Features
 * 50 Levels of Progression: Increasing speed, obstacle density, and AI difficulty.
 * Dynamic Biomes: Four distinct themes:
   * Sunflower Fields 🌻: Bright, high-visibility daylight.
   * Midnight Moon 🌙: Neon-lit night racing.
   * Sunset Rapids 🌅: Golden hour aesthetics.
   * Cosmic Challenge 💫: Reality-warping endgame visuals.
 * Combat Mechanics:
   * Quack-zooka (Forward): Stun opponents ahead of you.
   * Poop Traps (Rear): Drop obstacles to slow down trailing rivals.
   * Star Boosts: Collect glowing rings for massive speed bursts.
 * Physics-Based Racing: Drifting, "Beakie" leans, and river-buoyancy physics.
 * Mobile Responsive: Optimized touch controls and adaptive HUD for all screen sizes.
🎮 How to Play
Desktop
 * W / Arrow Up: Accelerate
 * A/D / Arrow Left/Right: Steer
 * Space: Jump (over logs or to initiate drift)
 * Shift / E: Use Item (Quack-zooka or Poop Trap)
Mobile
 * Left/Right Arrows: Steer
 * Rocket Icon: Jump
 * Poop Icon: Use Item
 * Auto-acceleration is enabled for mobile play.
🛠️ Technical Stack
 * Graphics: Three.js (WebGL)
 * Styling: Tailwind CSS
 * Framework: Vanilla JavaScript
 * Icons: Emoji-based procedural assets
